¿Qué es el Máster RIAtec?

Es un estudio de postgrado (máster propio) de la Universidad de Málaga, orientado a ofrecer una formación intensiva sobre las últimas tecnologías relacionadas con la Web, Internet y dispositivos móviles.

El temario está orientado al mercado laboral y se actualiza todos los años para reflejar los cambios tecnológicos.

El máster ofrece prácticas en empresas de hasta 1100€ al mes y hasta un año de duración. La mayoría de alumnos de pasadas ediciones han sido contratados tras el período de prácticas.

¿Qué tecnologías?

En el temario de esta edición se cubriran tecnologías tales como:

  • Web: HMTL5, JavaScript, jQuery, CSS3, MVC ASP .NET
  • Lenguajes: C#, Java
  • Aplicaciones móviles y tablets: Android, Windows Phone 8, Win RT, iOS
  • Acceso a datos: SQL Server, ADO. NET Entify Framework + LINQ
  • Cloud computing: Azure
  • Big data: Hadoop
  • ALM: git, mercurial
  • IDEs: Visual Studio 2012, Eclipse, Expression Blend
